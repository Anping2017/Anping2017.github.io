插值 指令 参数 修饰符

插值

  • 文本
    • {{msg}}--对接实例对象的属性值，双向绑定
      •                 //html
                        <div id="app">
                            {{ data }}  //对接app中data的值
                        </div>

                        //js
                        var app = new Vue({
                            el:"app",
                            data:{
                                data: "这里的值对接{{data}}"
                            }
                        })
                        
                        app.data = "这个值双向绑定{{data}}"
    • v-once--单次输出
      •                 //在元素中使用v-once,数据改变时不会更新
                        <p v-once>{{ data }}</p>
  • raw html--原始html输出
    •                     //为元素增加v-html="msg"属性
                    <p v-html="data raw html"></p>
  • Attribute属性值的变更
    • v-bind:attr
    • :attr
      •                 //为元素增加v-bind:attr = "data" 
                        <div v-bind:id = "data"></div>
                        //简写
                        <div :id = "data"></div>
