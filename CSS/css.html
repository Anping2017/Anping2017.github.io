    • 元素选择器(E)
    • 包含选择器(E F) -- 选择所有被E元素包含的F元素
    • 子代选择器(E>F) -- 选择所有作为E元素的子元素F
    • 相邻选择器(E+F) -- 选择紧贴在E元素之后F元素，元素E与F必须同属一个父级。
    • 兄弟选择器(E~F) -- 选择E元素后面的所有兄弟元素F，元素E与F必须同属一个父级。
